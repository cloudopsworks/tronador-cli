# Homebrew packaging

Homebrew packaging is generated by GoReleaser from `.goreleaser.yml`.
Do not use a separate generator for the cask/formula metadata.

The release pipeline uses the `homebrew_casks` section to derive metadata from
published release archives and checksums. Publishing to the tap is
automatically skipped unless `HOMEBREW_TAP_TOKEN` is available.

Primary validation:

```bash
goreleaser check
```
